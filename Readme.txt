1: Versão JAVA = java 17.

2:  Para utilizar o sistema basta acessar a classe "conexao" dentro do pacote "sql.biblioteca" e então mudar o "user" e o "password" que estão la para o usuario e senha do seu servidor.

3: importe os dados das tabelas com o arquivos CSV correspodentes a elas,a tabela "editora" vai importar o arquivo "editora.csv" e assim por diante

4: para ter acesso as funcionalidades do sistema,vc deve acessar ele com algum usuario e sua senha respectiva que estiverem salvos na tabela "usuario",lembrando que tem distinção entre usuario normal e admins,então para ter acesso total a tudo do sistema deve-se entrar com algum usuario que tenha o atributo "isAdmin" definido como "1" (admin)
